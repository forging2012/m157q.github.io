  • 演算法的重要性 - 你如何開平方根? SQRT(N)

    • Fast_inverse_square_root
    • 二分逼近法 
left, right = 0, n  
mid = (left + right) / 2.0  
while True:  
            if mid * mid > n:  
                right = mid  
            else:  
                left = mid  
            prev = mid  
            mid = (left + right) / 2.0  
            if abs(mid - prev) < 1e-6:  
                break  
  return mid
